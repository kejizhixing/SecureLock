【软件介绍】
安全锁是一款专业的微信/QQ隐私保护工具，通过文件级保护防止他人未授权访问您的聊天软件。
原程序备份为隐藏的.dat文件，用安全锁替换原程序，必须通过密码验证才能启动。

【支持系统】
- Windows 10 / 11 (64位)
- 需要安装 VC++ 运行库

【软件功能】
1. 文件级保护：原程序备份为.dat隐藏文件，用安全锁替换
2. 密码验证：双击被保护程序弹出密码框，验证后启动
3. 窗口隐藏：锁定时自动隐藏微信/QQ窗口
4. 空闲锁定：可设置空闲时间，超时自动锁定
5. 托盘管理：系统托盘图标，方便操作
6. 开机自启：支持开机自动启动
7. 进程监控：锁定时持续监控，防止绕过
8. 一键恢复：可随时恢复原程序

【安装方法】
1. 将 SecureLock.exe 复制到微信/QQ安装目录
2. 双击运行 SecureLock.exe
3. 首次运行设置密码

【使用教程】

一、首次设置
1. 双击运行 SecureLock.exe
2. 输入密码并确认（密码最少4位）
3. 点击「程序管理」添加要保护的程序
4. 选择微信/QQ的安装目录
5. 点击「保护」完成保护

二、日常使用
1. 解锁：双击托盘图标 → 输入密码 → 微信/QQ自动启动
2. 锁定：点击托盘菜单「立即锁定」
3. 启动程序：托盘菜单 → 快速启动 → 选择微信/QQ
4. 退出：托盘菜单 → 退出程序 → 输入密码确认

三、保护原理
1. 保护前：WeChat.exe (原程序)
2. 保护后：
   - WeChat.dat (原程序备份，隐藏)
   - WeChat.exe (安全锁副本)
3. 双击WeChat.exe → 弹出密码框 → 验证后启动WeChat.dat

【重置密码方法】
方法一：命令行重置
1. 按 Win+R 打开运行
2. 输入 cmd 回车
3. 输入以下命令：
   reg delete "HKCU\Software\SecureLock" /f
4. 重新运行 SecureLock.exe 设置新密码

方法二：删除配置文件
1. 打开以下目录：
   C:\Users\你的用户名\AppData\Roaming\SecureLock
2. 删除 Settings.ini 文件
3. 重新运行 SecureLock.exe 设置新密码

方法三：使用注册表编辑器
1. 按 Win+R 输入 regedit 回车
2. 导航到：HKEY_CURRENT_USER\Software\SecureLock
3. 右键删除整个 SecureLock 项
4. 重新运行 SecureLock.exe 设置新密码

【恢复原程序】
如果安全锁出现问题，可以手动恢复：
1. 打开微信/QQ安装目录
2. 显示隐藏文件（文件夹选项 → 显示隐藏文件）
3. 找到 WeChat.dat 或 QQ.dat
4. 删除 WeChat.exe（安全锁副本）
5. 将 WeChat.dat 重命名为 WeChat.exe
6. 或使用备份文件管理.bat一键恢复

【常见问题】

Q: 双击微信没有反应？
A: 检查是否已设置密码，或尝试重新运行SecureLock.exe

Q: 忘记密码怎么办？
A: 使用上述重置密码方法重置密码

Q: 如何完全卸载？
A: 1. 恢复原程序 2. 删除SecureLock.exe 3. 删除注册表项

Q: 保护后微信无法登录？
A: 检查.dat文件是否完整，或恢复原程序重新保护

Q: 开机自启如何取消？
A: 托盘菜单 → 取消自启动
